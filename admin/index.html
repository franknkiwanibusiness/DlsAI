Support Admin Mode
You are signed in as a Support Admin. You can view data, approve or reject listings, ban users, and add/remove sponsored & featured listings.

The following actions require Super Admin credentials:
  • Delete listings or users
  • Change user subscription plans
  • Export CSV data
  • Bulk approve all pending
  • Edit or remove listing images
  • Update the homepage hero image
Admin Access Control
Admin access is granted to emails in the ADMIN_EMAILS array at the top of the script block, or users where users/{uid}/isAdmin = true in Firebase.
Current admin emails (hardcoded)
Imgur Image Upload
The Imgur Client ID is defined in the IMGUR_CLIENT_ID constant at the top of the script block.

Replace YOUR_IMGUR_CLIENT_ID with your real Client ID from imgur.com/account/settings/apps. Image uploads are used in the Listing detail drawer.
